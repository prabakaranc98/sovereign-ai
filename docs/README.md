# docs — the knowledge home

Everything learnt, surveyed, curated — and the white paper as it takes shape toward publication. The scratchpad stays raw; INSIGHTS.md stays a high-bar distillery; this is where material gets *composed*.

Expected contents as the project matures:
- **Learnings** — study notes from the six frontier areas (feedback design, the judge layer, trace flywheels, memory, agentic RL, AI-native construction).
- **Surveys** — landscape maps, SOTA groundings, dimension × altitude maps, and the reading ledger that backs citations.
- **Curation** — organized reference collections worth sharing.
- **White paper** — working title space: *"What It Means to Own the Intelligence."* Positions earned by builds (`../src/`), landscape from the study passes.
- **Build write-ups** — each build in `src/` gets its report here, written to survive the persona filter (engineer, domain scientist, policymaker, founder, beginner).

Structure emerges from accumulation — no premature folder taxonomy. Nothing lands here from thin air: scratchpad → insights → composition here.
